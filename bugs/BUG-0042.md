# BUG-0042: Review mode shows only flashcards, not all 3 steps

**Status:** INVESTIGATING
**Severity:** P1
**Reported:** 2026-04-13
**Fixed in:** —
**Reporter:** Jiong

## Description
When tapping "Review" on a completed morning session, the review session is supposed to show all 3 steps (flashcards → image game → example questions). User reports it only shows step 1 (flashcards).

## Steps to Reproduce
1. Complete a morning session (all 3 steps)
2. Go back to Practice tab — morning shows as "Complete" with "Review" button
3. Tap "Review"
4. Expected: plays through flashcards → image game → example questions
5. Actual: may only show flashcards, or exit after step 1

## Investigation
- `SessionFlowViewModel(isReview: true)` creates all 3 steps correctly (lines 71-76)
- `loadWords()` skips session creation in review mode but still loads words (line 100-128)
- The `FlashcardStepView.onComplete` calls `vm.advanceToNextStep()` which should show `StepTransitionView`
- `StepTransitionView` auto-advances after 3 seconds via `Task.sleep`
- Possible issue: auto-advance in StepTransitionView may fire `onContinue` after the view is already dismissed
